# DevContainer Features

A collection of useful DevContainer features.

## Features

### UV

Installs the UV package manager with configurable version support.

```json
"features": {
    "ghcr.io/traverseda/devcontainer-features/uv:1": {
        "version": "latest"
    }
}
```

### Hello

A sample hello world feature.

```json
"features": {
    "ghcr.io/traverseda/devcontainer-features/hello:1": {
        "greeting": "hello"
    }
}
```
